Fitur

Semua yang kamu butuhkan
untuk monitoring serius

Bukan sekadar ping — WatchTower hadir dengan fitur enterprise yang biasanya hanya ada di solusi berbayar.

Flap Detection

Tahan notifikasi jika monitor UP-DOWN berulang dalam window waktu. Tidak banjir alert dari jaringan tidak stabil.

On-Call Schedule

Jadwal shift siaga tim. Alert otomatis dikirim ke channel yang sedang aktif bertugas — beda shift, beda penerima.

SLA Contract

Definisikan target SLA per layanan. Pantau sisa downtime budget dengan progress bar real-time. Cocok untuk laporan ke manajemen.

Incident Tracking

Insiden DOWN/UP tercatat otomatis. Tambah manual untuk gangguan non-teknis. MTTR, durasi, dan post-mortem RCA tersimpan.

Eskalasi Otomatis

Insiden tidak di-ack dalam X menit → alert dikirim ke channel eskalasi (manager, on-call tier 2). Bisa dikonfigurasi per monitor.

Status Page Publik

Buat halaman status di /status/slug. Widget embed, badge SVG, custom domain. Tidak butuh login.

Topology & Dependency

Visualisasi grafik dependency antar monitor. Jika parent DOWN → child skip notifikasi. Tidak spam alert saat DB mati.

Business Hours Aware

Konfigurasi jam kerja per hari. Alert routing berbeda di jam kerja vs luar jam kerja. Tidak bangunkan orang tengah malam.

REST API + Token

Akses data monitor via REST API dengan Bearer token. Integrasi ke dashboard internal, CI/CD pipeline, atau tools lain.

Pagespeed Monitor

Monitor skor Google PageSpeed Insights secara berkala — Performance, Accessibility, Best Practices, SEO, dan Core Web Vitals (CLS, FCP, LCP).

Audit Log

Semua aksi tercatat — siapa, kapan, dari IP mana. Monitor di-toggle, di-cek, dibuat, atau dihapus — semua masuk log.

Cara Kerja

Mulai Monitor dalam 3 Langkah

Langkah 1

Install & Deploy

Clone repo, jalankan composer install dan php artisan migrate. Atau pakai Docker Compose — selesai dalam 5 menit.

Langkah 2

Tambah Monitor

Klik tombol + Tambah Monitor di dashboard, pilih tipe, isi URL/host, atur interval dan channel notifikasi.

Langkah 3

Terima Alert

WatchTower cek tiap menit. Saat DOWN, alert langsung dikirim ke Telegram/WA/Email/Slack. Saat pulih, notifikasi recovery otomatis terkirim.

Instalasi

Deploy di Server Sendiri

Self-hosted — data kamu, privasi kamu. Tidak ada biaya subscription.

Server Langsung PHP 8.2 + MySQL 
# 1. Clone
git clone https://github.com/dani2159/uptime-server
cd uptime-server

# 2. Install
composer install --no-dev
cp .env.example .env
php artisan key:generate

# 3. Database
php artisan migrate
php artisan db:seed

# 4. Scheduler (crontab)
* * * * * php artisan schedule:run
Docker Compose Termasuk MySQL 
# 1. Setup
cp .env.example .env
# edit APP_URL dan DB_PASSWORD

# 2. Build & run
docker compose up -d --build

# 3. Init
docker compose exec app \
  php artisan migrate --force
docker compose exec app \
  php artisan db:seed

# Akses di http://localhost:8080
